Detailed Description

ROS 2 Lifecycle Node for managing the Piper TTS system.

This node wraps the libpiper C API to provide text-to-speech functionality as a ROS 2 action server. It downloads voice models from HuggingFace Hub and streams synthesized audio as AudioStamped messages.

See also
https://github.com/OHF-Voice/piper1-gpl
